# opresis
Scripts to scrape recurrence intervals for historic flood events. Several different options to gather historic rain data, including NWS PNS, SC-ACIS, and wpc tropical cyclone reports. Compares rain reports to ATLAS ARI estimates and grids, plots, and performs calculations. Can be used to rank historic events by area of various ARI threshholds. Can also be paired with crest data. 

As far as I know, this is a novel approach to plotting and ranking historic rain/flood events.
